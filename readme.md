## Controle Financeiro

Esse projeto tem por objetivo registrar as despesas e receitas de uma pessoa,
para um maior controle do seu dinheiro.

Projeto desenvolvido para treinar lógica de programação e modularização.

Funcionalidades 

- Adicionar receita
- Adicionar despesa
- Mostrar saldo
- Mostrar relatorio

Estrutura do Projeto

- lib/menu.py - responsavel pela interação com o usuario.
- lib/financeiro.py - responsavel pelas funções do sistema
- main.py - responsavel por controlar o fluxo principal do projeto

Tecnologias Utiizadas

- Python 3

Como executar

- Abaixe os arquivo do projeto
- Execute o arquivo main.py
